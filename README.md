# CommerceSystem
여러 상품을 구매하고 관리 할 수 있는 스파르타 커머스 시스템

## 구현 기능
1. 관리자 로그인 기능
   - 특정 이름, password입력 시 관리자 모드를 활성화 시킬 수 있습니다.
   - 로그아웃을 통해 일반 사용자모드로 돌아 갈 수 있습니다.
2. 관리자로 로그인한 경우 재고 관리를 할 수 있습니다.
   - 새로운 상품을 등록 할 수 있습니다.
   - 기존 상품을 제거 할 수 있습니다, 제거 시 장바구니에 있는 동일한 상품도 모두 제거됩니다.
   - 기존 상품의 수량을 조절 할 수 있습니다.
3. 상품 조회
   - 상품 전체 리스트를 조회 할 수 있습니다.
   - 상품의 특정 카테고리 리스트를 조회 할 수 있습니다 (1. 전자 제품 | 2. 주방 용품 | 3. 음식 | 4. 음료)
   - 원하는 금액 초과, 미만 상품들을 조회할 수 있습니다.
4. 장바구니
   - 상품 조회를 통해 불러온 상품중 원하는 상품을 장바구니에 넣을 수 있습니다 (재고 수량 이상으로는 담을 수 없습니다)
   - 담겨 있는 상품 리스트를 조회 할 수 있습니다.
   - 구매를 원치 않는 상품은 제거 할 수 있습니다.
   - 장바구니 상품 전체를 구매 할 수 있습니다.
5. 상품 구매
   - 장바구니를 통해 상품 구매가 가능합니다.
   - 총 구매 금액에 따라 고객님의 등급이 결정됩니다. (브론즈~플레티넘)
   - 등급에 따라 할인률이 다르게 적용됩니다. (0 ~ 15프로)
   - 결제 후 재고의 남은 수가 변경 됩니다.
   - 결제 후 고객님의 소유금액이 줄어듭니다.
   - 결제 후 장바구니는 자동으로 비워집니다.
